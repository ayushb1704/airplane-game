Vyorius Unity Selection Task

Overview :

This project implements a 3D open-world aircraft simulation in Unity, including realistic flight physics and interactive features as required in the assignment.

Environment :
• Desert open world
• Runway for takeoff and landing
• Lighting and atmospheric fog

Aircraft System :

• Rigidbody-based thrust system
• Speed-based aerodynamic lift
• Pitch, Roll, Yaw controls
• Stabilized physics tuning
• Center of mass adjustment

Camera System :

• Third-person follow camera
• Fixed nose camera view
• Camera switching functionality

Balloon Drop Feature :

• Balloon dropping mechanism
• Specific drop zones
• Visual feedback on successful drop
• Automatic cleanup of dropped objects

Photo Capture :

• Screenshot capture during flight
• Saved with timestamp

Controls :

• Throttle : W / S
• Pitch: Up / Down
• Roll: A / D
• Yaw: Left / Right
• Drop Balloon: Space
• Switch Camera: C
• Capture Photo: P

Unity Version :

6000.3.0f1
